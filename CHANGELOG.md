# Change Log

All notable changes to the "jandan-vscode" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [Unreleased]

## [0.3.0] - 2026-06-28

### Added

- Add Hot List view with 4-hour, content-specific, and best-ranking categories.
- Add NvZhuang, Tree Hole, and Q&A views with navigation and voting support.

### Changed

- Migrate project scaffold to Vite+ and upgrade dependencies.
- Modernize API requests with native fetch, JSON endpoints, cancellation, pagination, and shared types.
- Unify comment voting across supported views.

### Fixed

- Update Support view site names and URLs.

## [0.2.1] - 2023-07-28

### Fixed

- Fix bug with 'boring image' not being able to vote
- Fix bug with page scrolling not being restored when switching content
